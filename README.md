# Mica

Mica is an [ACP](https://agentclientprotocol.org)-native coding agent harness in Go.
Mica is a minimal, modern coding agent.

Built on [`go-acp-sdk`](https://github.com/carsonfarmer/go-acp-sdk) for the ACP protocol.

## Build

```bash
go build -o bin/mica ./cmd/mica
```

## Test

```bash
go test ./...
```
